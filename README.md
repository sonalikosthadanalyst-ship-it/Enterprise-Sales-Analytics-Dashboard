# Enterprise Sales Analytics & Business Intelligence Dashboard (2026 Q2)

## 📊 Project Overview
An end-to-end business intelligence pipeline integrating advanced SQL relational database querying with interactive Power BI analytical dashboards. This enterprise project transforms raw transactional data into actionable business insights, tracking critical metrics like revenue generation, product category performance, and regional sales distribution.

## 🛠️ Data Pipeline & Architecture
* **Data Extraction & Transformation (SQL Engine):** Structured relational database logic to clean raw tables, define formal schemas, and calculate automated revenue metrics dynamically (e.g., `Total_Revenue = Units_Sold * Unit_Price_INR`).
* **Data Visualisation & Modeling (Power BI):** Developed an interactive multi-dimensional dashboard that imports the processed SQL schemas to visually display complex metrics like vehicle model popularity, regional performance grids, and sentiment flags.

## 🧰 Tech Stack & Tools
* **Database Querying:** SQL (Relational Schema Design & Logic Generation)
* **Business Intelligence Tool:** Power BI Desktop
* **Script Integration Architecture:** Python (Pandas/NumPy) for automated schema prototyping

## 📂 Repository Contents
* `Sales_Analytics_Queries.sql` - The primary SQL production script containing data definition and logical computation queries.
* `Sales_Operations_Dashboard.pbix` - The production Power BI desktop application file containing full relational models and visualizations.
